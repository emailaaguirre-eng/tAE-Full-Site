# tAE Website

A modern, professional website built with Next.js, TypeScript, and Tailwind CSS.

## Color Palette

- `#ebf4f6` - Lightest (backgrounds)
- `#bdeaeb` - Light (accents)
- `#76b4bd` - Medium (primary actions)
- `#58668b` - Dark (headers, buttons)
- `#5e5656` - Darkest (text, footer)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js app directory with pages and layouts
- `/components` - React components
- `/public` - Static assets

## Features

- ✅ Modern, responsive design
- ✅ Mobile-first approach
- ✅ Smooth scrolling navigation
- ✅ Contact form
- ✅ Professional color scheme
- ✅ TypeScript for type safety
- ✅ Tailwind CSS for styling
- ✅ SEO optimized

## Build for Production

```bash
npm run build
npm start
```

## Customization

You can easily customize:
- Colors in `tailwind.config.ts`
- Content in component files
- Metadata in `app/layout.tsx`
